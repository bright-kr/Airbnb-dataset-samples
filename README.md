# Airbnb-dataset-samples

<h2>1001개의 Airbnb 숙소 리스팅으로 구성된 샘플 データセット</h2>

![Airbnb dataset header](https://github.com/luminati-io/Airbnb-dataset-samples/blob/main/airbnb-datasets.PNG)

1000개 이상의 숙소 리스팅으로 구성된 Airbnb データセット 샘플입니다. データセット는 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>이 무료 データセット에 포함된 데이터 포인트:</h2>

* ```property_id```: Airbnb의 숙소 리스팅에 대한 고유 식별자입니다
* ```name```: Airbnb 숙소 리스팅의 제목 또는 이름입니다
* ```url```: Airbnb 숙소 리스팅의 원본 URL입니다
* ```final_url```: 리디렉션 또는 수정 이후의 업데이트된 URL일 수 있습니다
* ```location```: 숙소의 일반적인 위치 정보입니다
* ```category```: Airbnb 숙소의 유형 또는 카테고리입니다
* ```price```: 숙소 대여 1박당 비용입니다
* ```image```: 숙소 이미지의 URL 또는 참조입니다
* ```description```: Airbnb 숙소를 설명하는 텍스트 세부 정보입니다
* ```guests```: 숙소가 수용할 수 있는 게스트 수입니다
* ```ratings```: 게스트 리뷰를 기반으로 한 평균 평점입니다
* ```reviews```: 숙소가 받은 리뷰 수입니다
* ```seller_info```: 숙소 소유자 또는 호스트에 대한 정보입니다
* ```pets_allowed```: 숙소에서 반려동물 허용 여부를 나타냅니다 (True/False)
* ```house_rules```: 숙소 소유자가 설정한 규칙 또는 가이드라인입니다
* ```amenities```: 숙소에서 제공되는 편의시설 또는 시설 목록입니다
* ```available_dates```: 숙소 예약이 가능한 날짜입니다
* ```lat```: 숙소의 위도 좌표입니다
* ```long```: 숙소의 경도 좌표입니다

그 외에도 훨씬 더 많은 항목이 포함됩니다.

이는 "Airbnb Properties Information (public data)" データセット에서 파생된 샘플 서브셋이며,
이 データセット에는 <b>11,000,000개 이상의 companies</b>가 포함됩니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, 또는 Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 방식 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 빈도: <b>Once, Daily, Weekly, Monthly, Quarterly, 또는 Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 강화: <b>요청 기반.</b>

<h2>Airbnb データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. 경쟁사 모니터링</h3>

경쟁사의 위치, 숙소 리스팅, 고객 평점 및 리뷰를 면밀히 모니터링합니다. 경쟁사를 능가할 수 있도록 리스팅과 가격을 최적화할 수 있는 영역을 파악합니다.

<h3>2. 브랜드 인식</h3>

자사 브랜드 및 경쟁사에 대한 고객 의견에 대한 인사이트를 확보합니다. 소비자 선호와 시장 트렌드의 변화를 감지하여, 브랜드 평판을 유지하거나 향상시키기 위해 전략을 조정할 수 있습니다.

<h3>3. 시장 인사이트</h3>

Airbnb에서 인기 있는 목적지와 새로 등록된 숙소를 발굴하여 업계 트렌드를 선도합니다. Airbnb 데이터를 활용해 심층 시장 분석을 수행하고, 머신러닝 이니셔티브를 추진하며, 혁신적인 오퍼링을 기획합니다.
